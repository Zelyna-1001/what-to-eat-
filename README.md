# 今天吃什么 PWA - 无氧/有氧/休息日版

这个版本按三种日类型生成饮食组合：

- 无氧日：早餐、午餐、晚餐、练后餐
- 有氧日：早餐、午餐、晚餐、练后餐
- 休息日：早餐、午餐、晚餐、轻练后餐/加餐

默认示例按女、21岁、165cm、62kg、每日预算22元、每日热量差450 kcal 估算：

- 无氧日：约1550 kcal，蛋白质120g，碳水145g，脂肪55g
- 有氧日：约1350 kcal，蛋白质110g，碳水110g，脂肪50g
- 休息日：约1150 kcal，蛋白质95g，碳水80g，脂肪45g

内置食物包括食堂一荤两素、米饭、全麦面包、玉米、红薯、牛奶、无糖豆浆、鸡蛋、香蕉、无糖酸奶，以及小番茄、海苔、魔芋果冻等减脂零食。

## 上传 GitHub Pages

把这些文件上传到仓库根目录：

- README.md
- app-macro-v2.js
- styles-macro-v2.css
- index.html
- manifest.json
- sw.js
- apple-touch-icon.png
- icon-192.png
- icon-512.png

不要上传 zip 压缩包。上传后在 Settings -> Pages 开启 GitHub Pages。

## 数据说明

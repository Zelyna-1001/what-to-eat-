# 今天吃什么 PWA

这是一个可以上传到 GitHub Pages 的纯前端网页 App，用来根据每日预算、训练日/休息日热量目标和食物库，随机组合早午晚餐。

## 功能

- 首次进入填写性别、年龄、身高、体重、每日预算、热量差。
- 支持训练日/休息日不同热量估算。
- 食物库可手动添加：热量、价格、蛋白质、碳水、脂肪、膳食纤维、适合餐次。
- 点击“随机组合”生成早午晚餐。
- 可以单独“换一个”早餐/午餐/晚餐。
- 可以把已吃餐次标记为“已吃/锁定”，之后重抽其他餐次时会保留。
- 支持导出/导入备份。

## 上传 GitHub Pages

把本文件夹里的所有文件上传到仓库根目录，不要上传 zip 压缩包。

需要上传：

- index.html
- styles.css
- app.js
- manifest.json
- sw.js
- icon-192.png
- icon-512.png
- apple-touch-icon.png
- README.md

## 注意

这个版本是纯前端本地 App，不会把照片或数据上传到云端，也不能直接调用 ChatGPT 自动识别照片。拍照分析营养元素需要后端/API 版本。
